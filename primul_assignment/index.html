Introducere în Front-End Development

Front-End Development este un domeniu de dezvoltare de software care se ocupă de crearea părții de prezentare a site-urilor web. Rolul programatorilor front-end este de a realiza în întregime tot ceea ce vede utilizatorul atunci când folosește un site web prin următoarele limbaje de codare și programare:

  • HTML
  • CSS
  • JavaScript
